# Output

This folder contains the generated output files from running the Water Pump RUL project on the dataset.

Contents include:
- Task 1 segmentation summaries
- Task 2 clustering summaries
- Task 3 Kadane analysis summaries
- `summary.json`
- `task1_segmentation_plots/` for segmentation visualizations
- `task3_kadane_plots/` for Kadane interval visualizations
